## 🎯 사용 방법

1. 해당 `.md` 파일 열기
2. 내용 복사 (`pbcopy` 사용 가능)
3. Perplexity/ChatGPT 등에 붙여넣기
4. 답변 받은 후 `analysis/` 폴더에 결과 정리

## 📝 작성 가이드

### 좋은 프롬프트 체크리스트
- [ ] 문제 상황 명확히 기술
- [ ] 구체적인 수치/데이터 포함
- [ ] 배경 맥락 제공
- [ ] 질문을 카테고리별로 구조화
- [ ] 대안 솔루션 나열 (평가 요청)
- [ ] 참고 자료/이론 언급

### 예시 구조
```markdown
# 질문 제목

## 문제 요약
[1-2 문단으로 핵심 설명]

## 현재 상황
- 데이터: ...
- 기술: ...

## 질문 사항
1. ...
2. ...

## 프로젝트 맥락
- 목표: ...
- 기술 스택: ...
```
